# 🚀 30 NumPy Practice Questions

## 📖 Overview

This repository contains solutions to **30 NumPy practice questions** implemented using Python and Jupyter Notebook.

The goal of this project is to strengthen the fundamentals of **NumPy**, which is one of the most important libraries used in:

- Data Science
- Data Analysis
- Machine Learning
- Artificial Intelligence
- Scientific Computing

Each question is solved in a separate Jupyter Notebook to ensure better organization, readability, and understanding.

---

# 🎯 Objectives

The primary objectives of this repository are:

- Learn NumPy fundamentals
- Understand array creation and manipulation
- Practice indexing and slicing
- Perform mathematical operations
- Learn statistical functions
- Work with matrix operations
- Understand aggregation techniques
- Prepare for Data Analyst and Data Scientist interviews
